The entire Varlock ecosystem is designed with security in mind.

## Security Features

- No more secrets in your codebase
- No more plaintext secrets on your local machine
- Biometric authentication for local development

Coming soon:
- Cloud-based secret storage for CI/CD and remote development
- Audit logs
