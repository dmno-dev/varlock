---
title: React Router
description: How to integrate Varlock with React Router for secure, type-safe environment management
---

[React Router](https://reactrouter.com/) (formerly Remix) applications do hybrid client/server rendering and it can be confusing as to where your code will be executed. Varlock gives you extra guardrails to ensure you will never leak a secret, and to provide better error messages when you do something wrong.

To use varlock in a React Router application, follow the instructions for our [Vite integration](/integrations/vite/).
